About ALTRADICAL

Page last updated: 28 February 2026

Introduction

ALTRADICAL is a game that I was planning to work on before developing The Semiradical.
It got cancelled at 17 February 2026.

General

Genre

Top-down shooter

Gameplay

You are playing as Yaromir. You will encounter cockroaches which are the enemies. You will also encounter Yaromir's clones which are the bosses.

Plot

Advanced cockroaches decided to start war against humans. However, a lectreido named Yaromir decides to sort out the conflict.
Additionally, in the world appeared Yaromir's versions from the other worlds.

Relation to The Semiradical

Both games have Yaromir as main character, have advanced arthropods and other general things. It's because ALTRADICAL is based on The Semiradical. Despite that, ALTRADICAL isn't canon.

Development

Concept

The concept originated lately, at the end of 2025 year. It was about a game mainly made for money.

Engine

The game would use own engine, probably written in C language with raylib library instead of ready-made engines.

Reasons of cancellation

  • Possible confusion with The Semiradical
  • No interest in top-down shooters
  • It was purely a commercial idea, an idea about a game created purely for money
